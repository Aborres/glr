uint32 utilities::Array< T >::put ( T &  element,

Adds element at the array index 'index'. Note that the runtime for this method is O(1) best case and O(n) worst case. Also, if index is greater than the current capacity of the array, the array will grow to accommodate the index (unless index > MAX_SIZE). This method will replace any element that is currently at 'index' without deletion, which may lead to memory leaks. Be sure to check the location is empty before adding an element with this method.

Parameters:
elementThe element to add
indexThe index in the array that the reference to element is to be put
Returns:
The index where the element was added, or -1 on error.

template<class T >
uint32 utilities::Array< T >::resize ( uint32  newSize) [inline]

Resize the array to the newly specified size. This function does not move any of the elements, it only increases the capacity of the array. Also, the array cannot be made larger than MAX_SIZE, and cannot be made smaller than the number of elements already in the array (including the empty space between elements in the array). If newSize is already the same size as the array, this function makes no changes to the array.

Parameters:
newSizeThe new size the array will be after the function executes.
Returns:
0 on success, or -1 on error.
